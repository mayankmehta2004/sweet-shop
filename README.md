# Sweet Shop Management System

A full-stack Sweet Shop Management System built using Node.js, Express, SQLite, and React.
The application supports user authentication, role-based authorization, inventory management,
and purchasing functionality.

---

## Tech Stack

- Backend: Node.js, Express
- Frontend: React
- Database: SQLite
- Authentication: JWT
- Testing: Jest, Supertest

---

## Features

- User registration and login
- JWT-based authentication
- Role-based access (Admin / User)
- Add, view, purchase, and restock sweets
- Inventory updates in real-time
- Backend API testing using Jest
